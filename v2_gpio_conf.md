# Mecanum f411re pinout

- Pins di direzione motori(FL-FR-BL-BR, A-B):
  
  - PC0..PC3
  
  - PC5
  
  - PC10..PC12

- Pins enable motori(FL-FR-BL-BR):
  
  - PA8...PA11

- Pins input encoders(FL-FR-BL-BR, A-B):
  
  - TIM2:
    
    - PA5
    
    - PB3
  
  - TIM3:
    
    - PA6
    
    - PA7
  
  - TIM4:
    
    - PB6
    
    - PB7
  
  - TIM5:
    
    - PA0
    
    - PA1

- Pins UART per ESP32
  
  - PC6(TX)
  
  - PC7(RX)
